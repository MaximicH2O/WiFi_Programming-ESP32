# WiFi_Programming-ESP32
Коротко о том, как загружать программу на esp32 через WIFI
Пошаговые действия для работы:
Внесите любые изменения в ваш код (например, добавьте мигание светодиодом).
В меню Arduino IDE выберите Инструменты > Порт > Сетевые порты.
Вы должны увидеть вашу ESP32, например, esp32-ota.local или с её IP-адресом . Выберите этот порт.
Нажмите кнопку Загрузка
Или загрузка через веб-интерфейс:
Шаг 1: Загрузка примера OTAWebUpdater по USB
В Arduino IDE откройте пример: Файл > Примеры > ArduinoOTA > OTAWebUpdater .
В этом скетче также пропишите свои ssid и password. Залейте его на ESP32 через USB. Откройте монитор порта, чтобы узнать IP-адрес устройства.
Шаг 2: Обновление через браузер
Откройте браузер на любом устройстве в той же сети и введите IP-адрес вашей ESP32.
Вы увидите веб-страницу с формой входа. По умолчанию логин и пароль — admin .
Для новой прошивки вам понадобится бинарный файл (.bin). В Arduino IDE скомпилируйте ваш проект и экспортируйте бинарный файл: Скетч > Экспортировать собранный бинарный файл .
На открывшейся веб-странице нажмите Choose File, выберите созданный файл .bin и нажмите Update. После загрузки ESP32 перезагрузится с новой прошивкой.
